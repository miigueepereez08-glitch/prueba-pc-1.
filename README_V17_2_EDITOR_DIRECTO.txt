KAYN ANDAMIOS PRO v17.2 - EDITOR DIRECTO

Cambios principales:
- Se elimina el flujo confuso de panel izquierdo "Estructuras" + panel derecho fijo de propiedades.
- Nuevo editor directo por medidas: largo, alto, ancho y tramo estándar.
- Nuevo editor directo por módulos: módulos de largo, niveles, profundidad y medida de módulo.
- El plano 2D pasa a ser el protagonista de la pantalla.
- Los módulos se pueden seleccionar y mover arrastrando directamente en el plano.
- Inspector flotante solo aparece cuando hay un módulo seleccionado.
- Despiece queda en panel desplegable, no estorba mientras se diseña.
- Mantiene compatibilidad con localStorage.kaynEditorModulos y localStorage.kaynCoreEngine.
- Mantiene sincronización con vista-3d.html.

Atajos:
- Rueda del ratón: zoom.
- Arrastrar vacío: mover plano.
- Arrastrar módulo: mover módulo con snap de 0,5 m.
- Supr/Delete: eliminar módulo seleccionado.
- Ctrl/Cmd + D: duplicar módulo seleccionado.
- Ctrl/Cmd + S: guardar proyecto.
