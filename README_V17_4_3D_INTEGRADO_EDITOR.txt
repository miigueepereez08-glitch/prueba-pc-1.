KAYN ANDAMIOS PRO v17.4 · 3D INTEGRADO EN EDITOR PRO

Cambios principales:
- Eliminado el flujo de 3D separado como herramienta principal.
- El Editor PRO ahora tiene un conmutador 2D / 3D dentro del mismo plano de trabajo.
- El 3D usa los mismos módulos creados en el editor 2D, sin abrir otra pantalla.
- No se añade Three.js ni carga pesada: render 3D ligero en canvas integrado.
- Los accesos del dashboard a 3D ahora llevan al Editor PRO en modo 3D.
- kayn-3d.html y vista-3d.html ya no contienen motores 3D: redirigen al Editor PRO por compatibilidad.

Uso:
1. Abrir Editor PRO.
2. Crear el andamio por medidas o módulos.
3. Pulsar 3D en el selector superior derecho del plano.
4. Volver a 2D cuando se quiera seguir editando.

Nota:
Esta versión prioriza estabilidad y limpieza. El 3D integrado es ligero y visual, pensado para comprobar el volumen sin salir del editor.
