KAYN ANDAMIOS PRO v18.0 - 3D REALISTA INTEGRADO EN EDITOR

Cambios principales:
- Mantiene un unico 3D dentro de editor-modulos.html.
- No hay visor externo operativo: kayn-3d.html y vista-3d.html redirigen al Editor PRO #3d.
- El 3D usa el mismo estado que el plano 2D: mismos modulos, misma seleccion y mismas posiciones.
- En vista 3D puedes seleccionar un modulo haciendo clic sobre el andamio.
- En vista 3D puedes mover el modulo seleccionado arrastrandolo, con snapping igual que en 2D.
- Si arrastras el fondo, gira la camara.
- Rueda del raton = zoom.

Mejoras visuales v18:
- Europeo/fachada con marcos tipo H, crucetas, barandillas intermedias, plataformas, rodapies y husillos.
- Multidireccional/ringlock con montantes, rosetas cada 0.5 m, largueros, diagonales, plataformas, rodapies y bases.
- Render 3D mas limpio, con seleccion naranja y piezas mas reconocibles.

Referencias de sistema utilizadas para orientar el modelado visual:
- Andamio europeo de Grupomaq: plataformas antideslizantes, rodapies y acceso por trampilla/escalera.
- ULMA BRIO multidireccional: acero galvanizado, sistema flexible, montantes con discos/rosetas y componentes certificados.

Nota:
Esta version sigue siendo ligera y sin librerias externas. Es una mejora realista integrada en canvas para no romper la app ni meter carga innecesaria.
